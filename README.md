## Enterprise IAM Engineering Labs

## Microsoft Entra ID • Conditional Access • PIM • Identity Governance • SSO • Automation

### Overview

This project demonstrates a hands-on Identity and Access Management (IAM) engineering lab environment built using Microsoft Entra ID, Microsoft Graph API, and PowerShell automation.

The lab simulates real-world enterprise IAM infrastructure and showcases how identity engineers design, secure, and automate identity systems using Zero Trust security principles.

The environment focuses on core IAM capabilities including identity governance, privileged access management, authentication security, application identity integration, and automated identity lifecycle management.

⸻

### Objectives

The goal of this project is to demonstrate practical IAM engineering skills including:  
    •    Identity Governance implementation  
    •    Conditional Access policy engineering  
    •    Privileged Identity Management (PIM) configuration  
    •    Enterprise application Single Sign-On (SSO) integration  
    •    Identity lifecycle automation using PowerShell and Microsoft Graph  
    •    Role-Based Access Control (RBAC) architecture  

These labs simulate the types of identity infrastructure deployed in modern enterprise cloud environments.

### Lab Modules

### Conditional Access Security Engineering

Design and implementation of layered Conditional Access policies to enforce Zero Trust authentication controls.  

Key features:  
    •    Multi-Factor Authentication (MFA) enforcement  
    •    Blocking legacy authentication  
    •    Device compliance requirements  
    •    Sign-in log validation and policy troubleshooting  

⸻

### Privileged Identity Management (PIM)

Implementation of Just-in-Time administrative access to eliminate standing privileged roles.  

Features:  
    •    Temporary role activation  
    •    MFA verification for privilege elevation  
    •    Time-bound administrative access  
    •    Privileged access audit logging  

⸻

### Identity Governance

Deployment of governance controls using Entitlement Management and Access Reviews.  

Features:  
    •    Access package creation  
    •    Time-bound resource access  
    •    Automated access expiration  
    •    Periodic access certification  

⸻

### Enterprise Application Single Sign-On

Integration of enterprise applications using SAML-based authentication through Microsoft Entra ID.  

Features:  
    •    Centralized authentication through Entra ID  
    •    RBAC-based application access  
    •    Conditional Access enforcement for applications  

⸻

### Identity Lifecycle Automation

Automation of the Joiner–Mover–Leaver (JML) identity lifecycle using PowerShell and Microsoft Graph.  

Automation includes:  
    •    Automated user provisioning  
    •    Security group assignment  
    •    License assignment  
    •    Access updates during role changes  
    •    Automated deprovisioning
